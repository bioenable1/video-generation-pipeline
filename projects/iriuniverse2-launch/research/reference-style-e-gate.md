## Production style cues

- **Tone:** Authoritative narrator, third-person corporate ("BioEnable introduces…")
- **Pacing:** ~140 WPM, complete sentences, no bullet-reading
- **On-screen text:** Short headline + 2–3 icon bullets (🔹 style in description = feature cards)
- **Visuals:** Real product/UI imagery + airport/security B-roll — NOT plain slides
- **Music:** Subtle corporate bed under narration
- **Length:** ~90–115 seconds total
- **SEO:** Keyword-rich YouTube title and description
