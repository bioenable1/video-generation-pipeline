# Optional local reference videos

This folder can hold a junction/symlink to your finished marketing videos for style reference.

**Windows (junction):**
```powershell
cmd /c mklink /J "reference\BioEnable-Videos" "C:\path\to\your\videos"
```

The sample project `projects/iriuniverse2-launch/` includes a complete worked example with `renders/final.mp4` — no local reference folder required.
